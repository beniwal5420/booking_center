# Technologies Involved

## Setup

This is newer, and so may have some gaps.

1. [Clone Repo](https://github.com/beniwal5420/booking_center)
2. Installations
   ```
   npm install
   ```
3. Start app
   ```
   npm start
   ```
4. Visit http://localhost:3000
